## 前提知識
* **Web基礎知識:** HTML (`.html`)、CSS (`.css`)、JavaScript (`.js`) の**ファイル拡張子**と、それぞれの**役割**を理解していること。
* **HTML/CSS/JavaScript:** 変数、セレクタ、タグなど、各言語の**基本的な概念**を理解していること。
* **Gitの基本:** コミット、ブランチの切り替え、クローンなど、**基本的なワークフロー**の知識があること。
* **ソフトウェアのダウンロード**方法がわかること
* **ショートカットキー**（Ctrl + S や Ctrl + C など）がわかること

これらの知識があるとスムーズに学習を進めることができますが、別になくても進められます。
